Robotics


Technoxian, World Robotics Championship, by International Robotics Council (IRC), 2017

The challenge was to build a quadcopter which is aerodynamically stable to achieve a successful flight. We built an indigeneous Light-Weight Sandwich Composite Multirotor System.

As the Avionics Technician on the project, I was responsible for designing, implementing and testing the quadcopter's propulsion system. Report
